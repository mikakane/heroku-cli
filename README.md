# Heroku Util

## Usage 

````
$ git clone https://github.com/mikakane/heroku-cli
$ cd heroku-cli
$ npm i 
$ heroku plugins:link .

````

## 実装済

- chatbox:open : open dashboard
- chatbox:jdbc : get jdbc url from DATABASE_URL

## 後で実装したいもの

### chatbox:php
